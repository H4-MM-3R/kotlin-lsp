# Kotlin Language Server (PoC)

This project is a proof of concept (PoC) for a Kotlin Language Server utilizing the [Kotlin Analysis API](https://kotlinlang.org/docs/analysis-api.html).
## Features

- **Diagnostics:** Real-time syntax and semantic error reporting.
- **Hover Support:** Display type information and documentation on hover.
- **Go to Definition:** Navigate to symbol definitions within the codebase.
- **Find References:** Locate all references to a symbol in the codebase.
- **Completion:** Context-aware code completion suggestions.
